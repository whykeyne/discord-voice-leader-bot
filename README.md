# Discord Voice Panel Bot

Готовый Discord-бот на Python, где управление голосовой комнатой идёт **через красивую UI-панель с кнопками** в отдельном текстовом канале.

## Что умеет

- Первый пользователь, который заходит в пустой войс, становится **лидером комнаты**.
- В отдельном текстовом канале бот создаёт **панель комнаты** с информацией:
  - название войса;
  - лидер;
  - сколько людей в войсе;
  - лимит комнаты;
  - открыта/закрыта комната;
  - список участников;
  - индикаторы mute/deafen/stream/camera.
- Управление идёт **через кнопки** на панели:
  - кикнуть участника из войса;
  - замутить / размутить;
  - заглушить / снять заглушение;
  - передать лидерство;
  - переименовать войс;
  - сменить лимит пользователей;
  - закрыть / открыть комнату;
  - обновить панель.
- Если лидер выходит из комнаты, лидерство автоматически переходит следующему участнику.
- При рестарте бот заново синхронизирует активные войсы и их панели.

## Файл `.env`

Скопируй `.env.example` в `.env` и заполни:

```env
TOKEN=your_bot_token_here
GUILD_ID=123456789012345678
CONTROL_CHANNEL_ID=0
CONTROL_CHANNEL_NAME=voice-control
STATE_FILE=panel_state.json
```

### Что значат переменные

- `TOKEN` — токен бота.
- `GUILD_ID` — ID сервера, нужен для быстрой синхронизации slash-команд.
- `CONTROL_CHANNEL_ID` — ID текстового канала, куда бот будет постить панели. Можно оставить `0` и потом указать канал командой `/setup_voice_panel`.
- `CONTROL_CHANNEL_NAME` — если `CONTROL_CHANNEL_ID=0`, бот попытается найти или создать канал с этим именем.
- `STATE_FILE` — JSON-файл для хранения состояния панелей.

- Move Members
- Mute Members
- Deafen Members
- Manage Channels
- Manage Roles

## Intents

В **Discord Developer Portal** включи:

- `SERVER MEMBERS INTENT`

`MESSAGE CONTENT INTENT` не нужен.

## Команды

Команд мало — только для настройки:

- `/setup_voice_panel` — указать текстовый канал, куда слать панели.
- `/force_sync_voice_panels` — пересобрать панели активных войсов.

## Как это выглядит

1. Пользователь первым заходит в пустой войс.
2. Он становится лидером.
3. В канале `#voice-control` появляется embed-панель этой комнаты.
4. Лидер нажимает кнопки на карточке и управляет комнатой.

## Важно

- Бот не сможет управлять участниками, у которых роль выше роли самого бота.
- Для lock/unlock используются permission overwrites канала.
- Если удалить текстовый канал панели, укажи новый через `/setup_voice_panel`.
- Для отображения старых панелей после перезапуска используются persistent views.
